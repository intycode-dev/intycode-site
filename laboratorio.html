Technical Testing Ground

Checking how Assembly code snippets look on the page:

; --- Simple Clear Screen Routine ---
CLEAR_SCREEN:
    MVII #$0000, R0    ; Move 0 into Register 0
    MVII #$0200, R4    ; Start of BACKTAB (Screen Memory)
    MVII #240,   R1    ; Number of tiles (20x12)
LOOP:
    MVO  R0, R4        ; Output R0 to memory address in R4
    INCR R4            ; Increment memory pointer
    DECR R1            ; Decrement counter
    BNE  LOOP          ; If counter != 0, repeat
    RETURN
